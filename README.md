# Rōmy

[intel.getromy.app](https://intel.getromy.app)

**Rōmy helps small nonprofits find new major donors at a fraction of the cost of existing solutions.**

![romy cover](./public/cover_romy.jpg)

## Features

- Multi-model AI support: OpenAI, Mistral, Claude, Gemini, and more
- Bring your own API key (BYOK) for cost-effective usage
- File uploads and document analysis
- Clean, responsive UI with light/dark themes
- Fully self-hostable and customizable
- Local AI support with Ollama

## Quick Start

```bash
npm install
echo "OPENAI_API_KEY=your-key" > .env.local
npm run dev
```

For detailed installation instructions, see [INSTALL.md](./INSTALL.md).
